# Portfolio Showcase

This repository contains a personal portfolio built with Vite, React, TypeScript, Tailwind CSS and shadcn-ui.

Local development

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Building for production

```sh
npm run build
npm run preview
```

Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

Deployment

You can deploy this project to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.).

Replace any project-specific settings and update metadata in `index.html` as needed.
